Api documentation

Send email

  • Endpoint: /api/sendEmail
  • Method: GET
  • Paramters required: email, template and api_link
  • email: email address to send the email to
  • template: a string containing the template.
  • api_link: We are gonna send a request to the supplied api_link to retrieve a json object which is used to the template compiling
  • Example:

    Template: 

    "my balance: [[Balance]] and my total count is: [[Total]]"

    Json object: 

    {
    "Balance": 500,
    "Total": 200
}

    Compiles to: 

    "balance: 500 and my total count is: 200"

    Note: we replace the name of the field inside [[ ]] with the value of the matching key in the json object
